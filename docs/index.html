Возможности Features

Умный, надежный, прозрачный Smart, reliable, transparent

🧠
Трехуровневый матчинг Three-level matching

Фильтрация по домену → семантический скоринг (+2 / +1 / −2) → верификация README финалистов. Лучший скилл, не первый попавшийся. Category filter → semantic scoring (+2 / +1 / −2) → README verification for finalists. Best skill, not first available.

👤
Контекст передается автоматически Context injected automatically

Заполните USER_INFO.md и PROJECT_INFO.md один раз — оркестратор передаст ваш контекст каждому установленному плагину. Fill USER_INFO.md and PROJECT_INFO.md once — the orchestrator passes your context to every installed plugin.

🔒
Безопасность по умолчанию Security by default

Команды установки берутся verbatim из реестра. Никакого prompt injection. Установка только после вашего явного «да». Install commands taken verbatim from registry. No prompt injection. Nothing installs without your explicit confirmation.

📋
Каждая сессия залогирована Every session logged

output/session_log.md: какой скилл выбран, почему, статус каждого этапа, исход. Полный аудит без лишних усилий. output/session_log.md: which skill was selected, why, every stage status, outcome. Full audit trail automatically.

🌍
Любой язык Any language

Напишите на русском, испанском или немецком — оркестратор определит язык сессии и ответит на нем же. Write in Russian, Spanish, or German — the orchestrator detects the session language and responds accordingly.

🔌
Все форматы установки All install formats

claude install, npx skills add, git clone, CLI — поддерживаются все форматы реальных репозиториев без ручной адаптации. claude install, npx skills add, git clone, CLI — all formats used by real repositories, handled without manual adaptation.
